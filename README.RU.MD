> "Как же хочется, чтобы пользоваться Open WebUI было чуть чуть удобнее!"

Ну, я здесь как раз за этим. Awesome WebUI - это **fork** [Open WebUI](https://github.com/open-webui/open-webui), который стремится улучшить опыт использования как для администраторов, так и для обычных пользователей. Вместо долгих вступлений лучше сразу перейти к нашим классным изменениям!

# Список Изменений
## Админ-панель
### #1. Список моделей
![МНЕ ЛЕНЬ ЭТО ПЕРЕВОДИТЬ, поэтому model list showing openai models: gpt 5, gpt 5.1, gpt 5.2, codex](preview/image.png)
```diff
+ Добавлена возможность выделять сразу несколько моделей
+ Клик средней кнопкой мыши по модели открывает её редактор в новой вкладке
+ Для мультивыделения добавлены массовые действия: смена иконки, имени, доступа и статуса вкл/выкл
```

### #2. Вкладка «Подключения»
<small>
* - "провайдер" здесь используется как синоним слова "подключение".
<br>¹ - функция пока не протестирована полностью, если найдёте баг — пожалуйста, сообщите об этом.
<br>² - использование SOCKS-прокси добавляет зависимость: `aiohttp-socks`.
</small>

![МНЕ ЛЕНЬ ЭТО ПЕРЕВОДИТЬ, поэтому preview of changes to the connections list](preview/image1.png)
```diff
+ Интерфейс разделён на 3 секции (OpenAI, Ollama, Дополнительные настройки)
+ Кнопки выровнены и сгруппированы слева для более понятной навигации
+ Base URL у подключений сделан кликабельным
+ Добавлено отображение префикса и тегов для каждого подключения
```
![МНЕ ЛЕНЬ ЭТО ПЕРЕВОДИТЬ, поэтому preview of changes to the connections list](preview/image2.png)
```diff
+ Добавлена поддержка прокси (HTTP/SOCKS4/SOCKS5) для подключений¹²
+ Добавлен «Дополнительный JSON», который добавляется к запросу
```

### #3. Вкладка «Awesome WebUI» (Authorization + SSO Management)
![preview of authorization settings in Awesome WebUI tab](preview/image3.png)
```diff
+ Добавлена отдельная админ-вкладка "Awesome WebUI" с разделами "Authorization" и "SSO Management"
+ Добавлены переключатели способов авторизации: регистрация, Email+Password signup, SSO-вход и создание аккаунтов через SSO
+ Добавлен контроль SSO-провайдеров для входа/регистрации с быстрыми действиями "All/None"
```
![preview of invite-only settings in Awesome WebUI tab](preview/image4.png)
```diff
+ Добавлены настройки invite-only доступа с выбором, кто может создавать коды (Admin Only / Selected Groups / All Users)
+ Добавлены параметры инвайтов: длина кода, срок жизни (пресеты + своя дата/время), префикс, reusable и max uses
+ Добавлено управление кодами приглашений (создать, скопировать, удалить)
```
![preview of sso management settings in Awesome WebUI tab](preview/image5.png)
```diff
+ Добавлен полный раздел SSO Management для настройки OAuth-провайдеров прямо из админки
+ Добавлены переключатели и редактирование OAuth-полей (Google, Microsoft, GitHub, Discord, OIDC, Feishu)
+ Добавлены расширенные параметры OAuth (merge by email, timeout, audience)
```
